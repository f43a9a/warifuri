Quick Start
-----------

Install warifuri:

.. code-block:: bash

    pip install warifuri

Initialize a new project:

.. code-block:: bash

    warifuri init project my-project

Add a task:

.. code-block:: bash

    warifuri init task my-task

List all tasks:

.. code-block:: bash

    warifuri list
